# Safetrack
Safetrack é um aplicativo de rastreamento em tempo real, pensado para segurança e monitoramento de familiares e amigos. Inspirado em aplicativos como Life360, o Safetrack permite que você saiba a localização de pessoas confiáveis, receba notificações sobre movimentações e organize grupos familiares de forma prática e segura.

## Funcionalidades
- Rastreamento de localização em tempo real usando Google Maps.<br>
- Notificações automáticas quando um membro entra ou sai de locais definidos.<br>
- Criação de grupos para adicionar familiares ou amigos próximos.<br>
- Interação com o mapa: visualize a localização de todos os membros, adicione pontos de encontro e marque locais importantes.<br>
- Armazenamento de dados em tempo real com Firebase Database.<br>

## Tecnologias Utilizadas
- Kotlin – Linguagem de programação do app.<br>
- Android Studio – IDE utilizada para desenvolvimento.<br>
- Firebase – Autenticação, banco de dados em tempo real e notificações.<br>
- Google Maps API – Exibição de mapas, interação em tempo real e marcação de pontos de interesse.<br>
